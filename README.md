# AI Builder Projects Repository 🏗️

A collection of AI and machine learning projects developed through the AI Builder program.

## 📁 Repository Structure

This repository contains multiple independent projects:

- **`proj0_sql_experimental/`** - SQL database experiments
- **`proj1_data_scraping_CVPR2024/`** - CVPR 2024 conference data scraping
- **`proj2_instagram_access_images/`** - Instagram image access tools
- **`proj3_enhanced_search/`** - Enhanced search algorithms

## 🐕 Coco Chat Project

**🚨 IMPORTANT: Coco Chat has been moved to its own dedicated repository for easier deployment and maintenance.**

### **📍 Find Coco Chat Here:**
**Repository:** [AI-Builder-Proj4-Deploy](https://github.com/CocoHusky/AI-Builder-Proj4-Deploy)

**Live Demo:** [coco-chat.ai-builders.space](https://coco-chat.ai-builders.space/) *(when deployed)*

### **What is Coco Chat?**
- ChatGPT-like interface with a **husky dog personality** 🐕
- AI Builder model integration
- **Personality guard rails** ensure consistent dog behavior
- Individual conversation management
- Dog-themed UI with emojis and behaviors

### **Features:**
- ✅ Dog personality that never breaks character
- ✅ Multiple AI model support
- ✅ Conversation saving/loading
- ✅ Personality validation and correction
- ✅ Deployed on AI Builder platform
